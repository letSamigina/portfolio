データパック

ワールドに入れるだけで動く仕組みとして、公開・配布しているものです。

  • QRコードを生成

    ダイアログからURLを登録すると、マップに綺麗に描画されるよう、ブロックを配置してQRコードを生成します。
    1枚の地図に収まるサイズのため、アイテムフレームに入れて壁に飾ることも可能です。

    • Minecraft
    • Java Edition
    • データパック
    紹介動画（YouTube） ソース（GitHub）

  • Roll und Flip（ダイス）

    ワールド内でダイスを振る機能を追加します。
    最大10個まで一度に振れ、出目の上限は1〜100です。
    結果は自分だけが確認できる「秘匿」と、全体に公開する「公開」を選べます。

    • Minecraft
    • Java Edition
    • データパック
    紹介動画（YouTube） ソース（GitHub）

  • XORを計算

    XORを計算するデータパックです。
    スコアボード ..factor_a と ..factor_b に値を入れ、/function smgn:exe を実行すると計算されます。

    • Minecraft
    • Java Edition
    • データパック
    紹介動画（YouTube） ソース（GitHub）

  • 快適なサバイバル

    サバイバルモードを快適にするための設定を追加するデータパックです。
    現在は天候の操作・死亡地点の確認などに対応しており、機能は順次拡張予定です。

    • Minecraft
    • Java Edition
    • データパック
    紹介動画（YouTube） ソース（GitHub）

開発ツール

運用や内容が個人向けに寄っているリポジトリは非公開としている場合があります。

  • mk_datapack

    Minecraft 用データパックを新規に作成するとき、必要なディレクトリやファイルを一括で生成します。
    毎回同じ構成を手で作る手間を省き、制作にすぐ入れるようにするためのツールです。

    • Common Lisp
    • Minecraft
    • データパック
    • 非公開
    リポジトリ非公開

  • mk_resourcepack

    Minecraft 用リソースパック向けに、必要なフォルダやファイルをワンクリックで用意します。
    テクスチャやモデルなどを追加していく前段階の土台づくりを自動化する用途です。
